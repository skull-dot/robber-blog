Basics of Terminal

While using Kali or Parrot we should know about some commonly used commands in terminal, because without terminal, hacking does not look like hacking.

$ sudo apt-get update && sudo apt-get upgrade

Everyday after opening terminal the first step is to update the packages and configuration files, if we do not do this so chances are there that we had to face ERRORS while running other scripts.

$ ping {IP-Address or site URL

ping (Packet Internet Groper) is the most commonly used tool for troubleshooting a network. Mostly it is used to check that host is up or not. It can also be used to check that you internet service is running or not.


EXAMPLE :-
$ ping 192.168.0.6

EXAMPLE :-
$ ping google.com

$ cd {directory}

To open a directory (Folder).


EXAMPLE :-
$ cd pages

$ ls

To list things in a directory (Folder).


EXAMPLE :-
$ ls

$ mkdir {directory name}

To make a directory (Folder).


EXAMPLE :-
$ mkdir pages

$ cp {file path you want to copy} {directory path where you want to copy}

To copy a file from one directory (Folder) to another.


EXAMPLE :-
$ cp /home/skull/Pictures/secret.txt /home/skull/Desktop

$ mv {file path you want to move} {directory path where you want to paste it}

To move one file from one directory (Folder) to another.


EXAMPLE :-
$ mv /home/skull/Pictures/secret.txt /home/skull/Desktop

$ rm -rf {file or folder name}

To delete or remove a file or directory (Folder).


EXAMPLE :-
$ rm -rf pages

EXAMPLE :-
$ rm -rf secret.txt

$ sudo ifconfig

To see your network interface configuration.

$ iwconfig

It is used to set the parameters of the network interface which are specific to the wireless operation

$ git clone {URL}

Github is a web-based platform used for version control. With github we can collaborate with other people on a project and there are many people on github providing wide range of softwares in which pentesting tools are also there. So, to download them on our device we can use command |git clone|


EXAMPLE :-
$ git clone https://github.com/rapid7/metasploit-framework

$ chmod {Permissions} {File-Name}

Command |chmod| is used to manage permissions of file. There are three types of permissions |Read|, |Write|, |Execute|.


EXAMPLE :-
$ chmod 777 secret.txt
|777 means that anyone can read, write and execute these file. We, should never set these permission.|

$ pwd

It prints the directory in which we are. [Print Working Directory]


EXAMPLE :-
$ pwd
======================
OUTPUT :-
$ /home/skull

$ touch

touch command is used to create files, this file can be anything.


EXAMPLE :- $ touch mad-skull.txt

EXAMPLE :- $ touch robber.zip

$ man {tool name}

man command is a very usefull command, it shows the user-manual of the tool.


EXAMPLE :- $ man ssh

$ nano {file name}

Nano is a text editor which is installed by default in linux.


EXAMPLE :- $ nano mad-skull.txt

$ sudo su

sudo is commonly used command, which means "superuserdo". This command is used when an operation requires root privieleges.


EXAMPLE :- $ sudo apt-get update

$ grep '{word}' {file name}

Find text in a file. The grep command searches through many files at a time to find a piece of text you are looking for.


EXAMPLE :- $ grep 'non-sense' mad-skull.txt

$ passwd {username}

Command passwd is used to change password of the user.


EXAMPLE :- $ passwd skull

$ chown {username} {filename}

This command is used to change the ownership of a file/folder or even multiple files/folders for a specified user/group.


EXAMPLE :- $ chown beast mad-skull.txt

EXAMPLE :- $ sudo chown beast robber.txt

$ wc {argument} {filename}

This command is used to see file contents like numbers-of-lines, words, bytes, characters.


wc -l : Prints the number of lines in a file.
wc -w : prints the number of words in a file.
wc -c : Displays the count of bytes in a file.
wc -m : prints the count of characters from a file.
wc -L : prints only the length of the longest line in a file.

EXAMPLE :- $ wc -l mad-skull.txt
======================
OUTPUT :- 453 mad-skull.txt

$ history

It shows the list of our previously used commands.


$ clear

Clears the terminal.


$ poweroff

Shutdown the PC/Laptop from terminal.


$ reboot

Restart the PC/Laptop
