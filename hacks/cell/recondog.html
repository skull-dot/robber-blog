ReconDog is OSINT (Open Source Intelligence) tool, also know as a ' Reconnaissance Swiss Army Knife '. Today we are going to use ReconDog to gather every possible information of an IP-Address.


$ apt-get update && apt-get upgrade

This will upgrade the repositories, it is important because sometimes we face errors if the repositories are not updated.

$ apt-get install python

this wil install latest version of python.

$ git clone https://github.com/s0md3v/
ReconDog

This will clone [Download] the ReconDog on your device for further use.

$ cd ReconDog

Enters into the ReconDog directory

$ python3 -m pip install -r requirements.txt

'pip' is the package installer for python, pip will install package required to run ReconDog.

$ python dog

This will execute the dog which is a python script.

=> ReconDog menu will appear

To gather information choose option '0', and then enter the IP-Address
