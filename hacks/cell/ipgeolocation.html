IPGeoLocation is a python script used to gather location of IP-Address.


$ apt-get update && apt-get upgrade

This will upgrade the repositories.

$ apt-get install python

If you have already installed python, then also run this command to confirm because this tool use Python 3.x

$ git clone https://github.com/maldevel/IPGeoLocation

This will install IPGeoLocation.

$ cd IPGeoLocation

Enters into the IPGeoLocation directory.

$ pip3 install -r requirements.txt

This will install pacakages required to run IPGeoLocation tool.

=> How to use IPGeoLocation ?

After installation run these command.

$ python IPGeoLocation.py -t xx.xx.xx.xx

[xx.xx.xx.xx == IP-Address]
