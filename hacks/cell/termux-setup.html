Termux-Setup

×

Termux is a terminal emulator with linux enviroment. In termux we can install script and tools execute them in bash and zsh shell, if you don't have a laptop or P.C then also you can perform wide range of tasks through termux.


=> Install termux

Termux is available on play-store you can download it from there, but termux will not support 'Android 6' it's support was dropped since '2020.01.01'. So, if you want to use termux then you should have a Android version higher than 6.

=> Setting up storage

after installing it first we need to setup storage. So, for that the command is.
$ termux-setup-storage

$ apt-get install && apt-get upgrade

This will upgrade the repositories.

=> Installing packages <=

For installing packages in termux we can use two commands
$ pkg install {package name}
$ apt-get install {package name}
and to search packages in database
$ pkg search {keywords}
Now we are goint to install some packages.

$ apt-get install python

This will install latest stable python version on your device.

$ apt-get install python2

This will install python 2.7.18-3 on your device.

$ apt-get install git

This will install package git which will be used to install tools from github.

$ apt-get install neofetch

By typing '$ neofetch' some information will be printed on your screen of your device with a cool banner.
