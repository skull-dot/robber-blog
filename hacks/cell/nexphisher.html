NexPhisher is a phishing tool which contain 30 phishing templates with 4 port-forwarding opitons and it's user-interface is easy for beginner.


$ apt-get update && apt-get upgrade

This will upgrade the repositories

$ git clone https://github.com/htr-tech/nexphisher

This will install nexphisher tool.

$ cd nexphisher

Enters into the nexphisher directory.

$ bash tmux_setup

This will install packages required to run nexphisher.

$ bash nexphisher

This will execute the nexphisher script.

=> Choose the option you want.

NOTE -:- Sometimes pages are broken or not written properly which can make your attack suspicious to victim, so if you are performing phishing attack on someone check the page.

=> Choose port-forwarding

SUGGESTION -:- Use ngrok for port forwading and turn on hotspot to get link because in mobile device ngrok gives link if hotspot is turned on.

=> Send link to victim

Information gathered from the victim will be displayed on the screen and also it will be saved in logs directory.

=> See stored information in logs directory.

=> Open termux


$ cd nexphisher


$ cd logs


$ ls


$ cat {file name with extension}
EXAMPLE -:- $ cat instagram.log
